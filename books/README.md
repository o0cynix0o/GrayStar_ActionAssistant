# Book Files

Grey Star Action Assistant does not include the Project Aon book HTML files.

Download the standard multi-page HTML ZIPs from Project Aon for your own personal use, then extract them here:

```text
books/
  gs/
    01gstw/
    02tfc/
    03btng/
    04wotw/
```

Each book folder should contain `title.htm`, section files such as `sect1.htm`, and the book's image/support files.

See `../docs/INSTALL_PROJECT_AON_BOOKS.md` for links and examples.
