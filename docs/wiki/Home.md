# Grey Star Action Assistant Wiki

Welcome to the project wiki for **Grey Star Action Assistant**.

This wiki is the player-facing companion to the main repository. It explains how to use the local web assistant while playing the **World of Lone Wolf / Grey Star** gamebooks.

## What This Project Is

Grey Star Action Assistant is a local Python web app that acts like a digital Action Chart and play aid. It keeps the book visible on the left and the assistant on the right.

It helps with:

- character sheet state
- Endurance, Willpower, Combat Skill, and Nobles
- inventory, Backpack Items, Special Items, Weapons, and Herb Pouch Items
- Magick powers and Willpower costs
- random number rolls from 0 to 9
- combat setup and combat round handling
- section effects and guided section support
- death, rewind, book completion, repeat-book flow, and saves
- achievements and achievement notifications

It is designed to support the books, not replace them.

## Current Scope

Current app support is strongest for:

- **Book 1: Grey Star the Wizard**
- **Book 2: The Forbidden City**
- **Book 3: Beyond the Nightmare Gate**
- **Book 4: War of the Wizards**

All four Grey Star books now have assistant rules, achievements, strategy guides, and green dry-run playtest reports.

The app is ready for public release packaging. For download, backup, import, and layout notes, see the [Public Release Guide](Public-Release-Guide).

## Project Rule

Book rules always win.

If the app has a convenience behavior and the book text says something else, follow the book. The assistant is meant to preserve the text's mechanics, not smooth them away into house rules.
