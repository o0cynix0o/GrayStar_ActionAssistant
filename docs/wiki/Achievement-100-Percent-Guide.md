# 100 Percent Achievement Guide

This is the cleanup plan for unlocking every Gray Star achievement in the assistant.

The final playtest unlocked `102/102` achievements. You can repeat the same plan yourself, but it is worth knowing up front: this is not meant to be one perfect heroic journey. It is a completionist sweep.

The important rule is simple:

Book 1 needs two runs. Books 2, 3, and 4 can each be cleaned up with one extended run.

## The Short Version

To reproduce the 100% achievement result:

- Finish Book 1 once without the Sorcery shield route before the Kleasa result.
- Replay Book 1 for the Sorcery shield and the missed side prizes.
- Do one extended Book 2 sweep.
- Do one extended Book 3 sweep.
- Do one extended Book 4 sweep.

The tested result ended with:

- Book 1: `31/31`
- Book 2: `25/25`
- Book 3: `23/23`
- Book 4: `23/23`
- Total: `102/102`

## Why Book 1 Needs Two Runs

Book 1 has one important order trap.

You want **No Shield, No Problem** before you replay for **Shield Raised**.

That means your first Book 1 completionist run should reach the Kleasa fight without the shield route. Once that achievement is safely unlocked, you can repeat Book 1 and go raise the Sorcery shield.

Do it in this order:

1. Main no-shield finish.
2. Shield and side-prize replay.

That order keeps the cleanup clean.

## Book 1 Run 2: Shield And Side-Prize Replay

Repeat Book 1 with the same character. Your achievements stay unlocked, but Endurance and Willpower reset for the new attempt.

For the replay, make sure you hit these targets:

- Raise the Sorcery shield before the Kleasa fight.
- Survive the Kleasa result after using the shield.
- Mix Ezeran Acid.
- Take the Redeemer item route.
- Succeed at the ravine jump.
- Trigger Willpower Burnout if it has not already unlocked.

## How To Keep The Cleanup From Getting Messy

Use Repeat Book instead of starting a totally new save when you are cleaning up a book. Repeat Book keeps the same character and achievement profile, but resets the current book attempt so you can try a different route.

Do not worry if a straight story run only unlocks about half the achievements. The final story playtest unlocked `51/102`, which is perfectly normal.

For combat achievements, use the Combat tab. The assistant records combat history, and the achievements read from that history.

For item achievements, actually take the item or visit the section that proves you found it. The assistant can backfill achievements from durable save history, but it still needs evidence.

For Book 1, remember the big rule: no-shield first, shield replay second.

Once you do that, 100% is viable.
