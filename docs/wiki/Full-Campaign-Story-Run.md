# The Full Campaign Story Run

> *"It is a grim, chill morning, your last on the Isle of Lorn. With quaking heart, you enter the sacred temple of Amida."*
>
> — Book 1, Section 1

This is the clean, one-character journey through the whole Grey Star series. One Grey Star. Four books. No save shuffling, no character resets, no chasing every achievement on the map — just the story as a story, played the way a first-time reader would play it. It starts on that grim chill morning above, and it ends with the Wytch-king finally answered, four books later.

Use this page when you want to **prove the campaign flow works the way a real player expects**: make a new character on the Isle of Lorn, finish Book 1, continue into Book 2, keep going into Book 3, and carry the same Grey Star all the way to the end of Book 4.

This is **not** a 100% route. The chase-every-achievement work is what the [100% Achievement Guide](Achievement-100-Percent-Guide) is for. This page is the spine — the playable journey the rest of the strategy work is built on top of.

## What This Run Proves

The story run is the campaign in its simplest, cleanest shape. It quietly confirms all the things that matter when you're actually playing the saga end-to-end:

- a new character can be created cleanly
- the same character carries from book to book without losing themselves
- each book's completion screen appears at the right time
- the continue-to-next-book rules apply correctly
- Endurance, Willpower, Combat Skill, inventory, completed books, and achievements all survive the handoff
- Book 4 ends the campaign without leaving a death or failure screen active

When we ran this end-to-end ourselves, the route below finished all four books with the same character and ended with Books 1, 2, 3, and 4 marked complete. That's the result you're after. Everything else on this page is the route, the setup, and the small things that make reproducing it painless.

## Character Setup

To match the run we used in testing, start a new Book 1 character with:

- Name: anything you like — Grey Star is the wizard, you're the player
- Starting book: **1**
- Lesser Magicks: **Sorcery**, **Enchantment**, **Elementalism**, **Alchemy**, **Prophecy**
- Shianti Gift: **Jewelled Dagger**

When we tested, we used high opening rolls: Combat Skill **19**, Endurance **29**, Willpower **29**. You do not have to match those numbers perfectly to play the campaign — the route is forgiving on the edges — but if you're trying to reproduce the test as closely as possible, stronger starting rolls make the journey friendlier. Roll honestly; reroll if your conscience and the dice can both look the other way.

## How To Play It

Start at Book 1, Section 1, and follow the route below. When a book completes, use the Assistant's **continue** option instead of manually jumping to the next book. That handoff is part of what this run is meant to prove works, and short-circuiting it defeats the point.

After Book 1, choose **Psychomancy** as the extra Lesser Magick. After Book 2, aim for a strong Book 3 Willpower roll on the handoff. After Book 3, choose all five **Higher Magicks** for Book 4. Yes, all five. You're going to need them.

Use the Assistant's buttons whenever a section offers them. Use the Combat tab for preset fights. Use the roll helper when the section calls for a random number. The route is much, much easier to reproduce when the app is handling the fiddly bookkeeping and you're just making the decisions.

## What The Final Run Looked Like

When we ran the route end-to-end, it finished like this:

- Book 1: completed at Section 350
- Book 2: completed at Section 310
- Book 3: completed at Section 350
- Book 4: completed at Section 360
- Final completed books: 1, 2, 3, 4
- Story-run achievements unlocked: **51 / 102**

That achievement count is normal. A straight story run is not supposed to collect everything — the Grey Star books were designed with mutually exclusive choices and side targets that no single playthrough can hit. Fifty-one out of a hundred and two is the *correct* output of a clean, story-first first read. If you want the other half, that's what the [100% Achievement Guide](Achievement-100-Percent-Guide) is for. The completionist sweep is built on top of this story run, not in place of it.

## Tips For Repeating The Run

A handful of small things that make the difference between a route that reproduces cleanly and a route that goes sideways at Section 200-something:

Use the Assistant's buttons whenever a section offers them. If a section has a Meal button, a potion button, an obvious automation control — use it. Do not manually adjust a stat if the section has an obvious automation for it. The buttons are tested. Hand-edits aren't.

Use the Combat tab for preset fights. The route is much easier to reproduce if the app is handling the special fight rules — Staff use, mental combat, shielded combat, survival rounds, all of it. The presets know the wrinkles.

When a book ends, use the **completion screen**. Don't skip past it, don't jump to the next book manually. The completion flow is the part this run is really meant to prove, and the only way to prove it is to use it.

If Grey Star dies, use **Repeat** for a bad fight roll and **Rewind** if the mistake happened in the previous section. Both are forgiving in ways the original gamebooks never were.

## Final Word

This run is the campaign spine — the proof that one Grey Star can carry a single thread from the temple of Amida on the Isle of Lorn to the wall of fire on the far side of Shadaki, through four books, with the same character intact at the end. Once it works, you can start caring about the shiny cleanup work. Until it works, nothing else really matters.

If you've never played the series before, do this run first. Take it slow. Let the books be books before you start treating them as a checklist. The achievements aren't going anywhere — they'll still be there when you come back for the [100% Achievement Guide](Achievement-100-Percent-Guide) sweep.

Good luck out there, Wizard. The road from Lorn is long, but it's a good one.
