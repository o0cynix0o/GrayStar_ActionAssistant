# Full Campaign Story Run

This is the clean, one-character journey through the whole Gray Star series.

Use this page when you want to prove the campaign flow works the way a real player expects: make a new character, finish Book 1, continue into Book 2, keep going into Book 3, and carry the same Gray Star all the way to the end of Book 4.

This is not a 100% route. It is the "play the story straight through" route.

## What This Run Proves

The story run checks the big stuff that matters when you are actually playing:

- new character creation works
- the same character carries from book to book
- each book completion screen appears at the right time
- continue-to-next-book rules apply correctly
- Endurance, Willpower, Combat Skill, inventory, completed books, and achievements survive the handoff
- Book 4 ends the campaign without leaving a death or failure screen active

In the final playtest, this run finished all four books with the same character and ended with Books `1`, `2`, `3`, and `4` marked complete.

## Character Setup

To match the tested story run exactly, start a new Book 1 character with:

- Name: anything you like
- Starting book: `1`
- Lesser Magicks: `Sorcery`, `Enchantment`, `Elementalism`, `Alchemy`, `Prophecy`
- Shianti Gift: `Jewelled Dagger`

The playtest used high opening rolls: Combat Skill `19`, Endurance `29`, and Willpower `29`.

You do not have to match those numbers perfectly to play the campaign, but if you want to reproduce the test as closely as possible, stronger starting rolls make the route more forgiving.

## How To Play It

Start at Book 1, section `1`, and follow the route below. When a book completes, use the assistant's continue option instead of manually jumping to the next book.

After Book 1, choose `Psychomancy` as the extra Lesser Magick.

After Book 2, the tested handoff used a strong Book 3 Willpower roll.

After Book 3, choose all five Higher Magicks for Book 4.

## Tips For Repeating The Run

Use the assistant's buttons whenever a section offers them. Do not manually adjust a stat if the section has an obvious automation button.

Use the Combat tab for preset fights. The route is much easier to reproduce if the app handles the special fight rules.

When a book ends, use the completion screen. That is the part this run is really meant to prove.

If Gray Star dies, use Repeat for a bad fight roll and Rewind if the mistake happened before the section.

This run is the campaign spine. Once it works, you can start caring about the shiny cleanup work.
